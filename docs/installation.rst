Installation
============

You can install dwc_tools using pip:

.. code-block:: bash

   pip install dwc-tools

Development Installation
------------------------

To install dwc_tools for development:

.. code-block:: bash

   git clone https://github.com/7yl4r/dwc_tools.git
   cd dwc_tools
   pip install -e ".[dev,docs]"

This will install the package in editable mode along with development and documentation dependencies.

Requirements
------------

* Python 3.8 or higher
